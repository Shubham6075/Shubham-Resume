## ğŸ”¹ LINKED LIST TEMPLATE

class ListNode {

        this.val = val;
        this.next = null;
    }

### Reverse Linked List

```java
public ListNode reverseList(ListNode head) {
    ListNode prev = null;

    while (head != null) {
        ListNode next = head.next;
        head.next = prev;
        prev = head;
        head = next;
    }
    return prev;
}
```
